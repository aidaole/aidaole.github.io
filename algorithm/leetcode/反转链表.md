# 24. 反转链表

给定单链表的头节点 head ，请反转链表，并返回反转后的链表的头节点。

循环解法：
```java
class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode pre = null;
        ListNode p = head;
        ListNode q = head;
        while(p != null) { // 当老链表头为null，结束
            // q 先指向p下一个节点，避免链表断裂
            q = p.next;
            // p 的next可以断开，指向已经调换的头
            p.next = pre;
            // pre 移动成为新链表头
            pre = p;
            // p 指向q，老链表头
            p = q;
        }
        return pre;
    }
}
```

递归解法：
```java
class Solution {
    public ListNode reverseList(ListNode head) {
        if(head == null || head.next == null){
            return head;
        }
        ListNode p = reverseList(head.next);
        head.next.next = head;
        head.next = null;
        return p;
    }
}
```
